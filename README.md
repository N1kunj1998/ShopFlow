# E-Commerce App

Simple full-stack Next.js application with API routes.

## Quick Start

```bash
npm install
npm run dev
```

The app runs on http://localhost:3000

- Frontend: http://localhost:3000
- API: http://localhost:3000/api

## Project Structure

```
├── app/
│   ├── api/        # API routes (backend)
│   └── page.tsx    # Frontend pages
```
